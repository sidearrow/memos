# AWS 認定試験学習メモ CloudFront 編

## 概要

- エッジロケーションにコンテンツがない場合、データはオリジンサーバーから取得する
- S3 のコンテンツを特定のユーザーに配信する場合、 プライベートに設定された S3 バケットにオリジンアクセスアイデンティティ（OSI）を利用してアクセスし、署名付き URL を利用して配信する
